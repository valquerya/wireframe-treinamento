Perguntas Frequentes
Como funciona a carreira Zabbix?▾
A carreira Zabbix é composta de 4 níveis de certificações que iniciam do Certified User até o Certified Expert. É uma trilha que a partir do terceiro nível, o Professional, é necessário cumprir o nível anterior para prosseguir. Apenas User e Specialist não têm requisitos.
Como posso me inscrever nos treinamentos?▾
Depois de selecionar um curso que atenda às suas necessidades, você poderá solicitar sua inscrição via formulário. Assim que recebermos sua solicitação, a equipe de vendas entrará em contato com você, normalmente em um dia útil.
Preciso atender pré-requisitos para participar?▾
Para cursos de nível inicial, não é necessário conhecimento prévio porque nós te guiamos desde os primeiros passos. Para outros, é necessário conhecimento prévio em determinadas tecnologias. Consulte a página do treinamento para verificar o pré-requisito de cada um.
O treinamento dá direito ao exame?▾
Adquirindo o treinamento oficial, o participante ganha um voucher para realizar o exame. Quando apto na prova, conquista a credencial de certificação, seguindo os requisitos do fabricante.
Os treinamentos são online ou presenciais?▾
Prioritariamente online ao vivo. Para empresas, é possível agendar turmas fechadas exclusivas, online ou presenciais na sua região.
Qual a diferença entre os cursos?▾
A trilha de treinamento Zabbix é um programa desenvolvido para participantes com diferentes habilidades e necessidades. Existem requisitos para alguns cursos — por exemplo, para receber o certificado de Profissional, é obrigatório ter concluído o de Especialista.

Perguntas sobre a Carreira Zabbix
Como funciona a carreira de certificações Zabbix?▾
A carreira Zabbix é composta por 4 níveis de certificações: Certified User (ZCU), Certified Specialist (ZCS), Certified Professional (ZCP) e Certified Expert (ZCE). ZCU e ZCS não exigem pré-requisitos. Para o ZCP é necessário ter o ZCS e para o ZCE é necessário ter o ZCP, sempre na mesma versão do Zabbix.
Por onde devo começar?▾
Para quem está entrando no Zabbix agora, o ZCU é uma boa introdução à plataforma. Para quem já tem alguma experiência ou quer ingressar diretamente na carreira técnica, o ZCS é o ponto de partida mais comum.
O exame já vem incluído nos treinamentos?▾
Sim. Nos treinamentos ZCS, ZCP e ZCE o exame de certificação está incluído. Ao final do treinamento, o participante recebe um voucher para agendar a prova. Sendo aprovado, a certificação oficial é emitida diretamente pelo fabricante Zabbix.
As provas são difíceis?▾
As provas do fabricante são exigentes e testam tanto o conhecimento teórico quanto a aplicação prática. O treinamento oficial é projetado para preparar o aluno para a prova, mas dedicação e estudo complementar fazem diferença.
Posso fazer o ZCP sem ter feito o ZCS com a Selbetti?▾
Sim. O pré-requisito é a certificação ZCS, independente de onde foi obtida. Se você já tem o ZCS de outra escola ou pelo exame avulso, pode ingressar diretamente no ZCP.
Os treinamentos são online ou presenciais?▾
Os treinamentos ZCU, ZCS e ZCP são realizados online ao vivo com instrutor. O ZCE é presencial. Para empresas, é possível contratar turmas fechadas exclusivas em qualquer formato.
O que muda entre versões do Zabbix?▾
O Zabbix lança novas versões LTS periodicamente com mudanças significativas na plataforma. As certificações são vinculadas à versão — um ZCS 6.0 não equivale a um ZCS 7.0. Para quem já é certificado em versão anterior, os cursos de Upgrade atualizam a certificação.
Como funciona o agendamento do exame avulso?▾
Para quem adquire apenas o exame (sem o treinamento), o agendamento é feito pela plataforma EAD Selbetti após a compra. A prova é realizada em data combinada com o candidato.
