Perguntas Frequentes
Como funciona a trilha Zabbix?▾
A trilha Zabbix é composta de 4 níveis de certificações que iniciam do Certified User até o Certified Expert. É uma trilha que a partir do terceiro nível, o Professional, é necessário cumprir o nível anterior para prosseguir. Apenas User e Specialist não têm requisitos.
Como posso me inscrever nos treinamentos?▾
Depois de selecionar um curso que atenda às suas necessidades, você poderá solicitar sua inscrição via formulário. Assim que recebermos sua solicitação, a equipe de vendas entrará em contato com você, normalmente em um dia útil.
Preciso atender pré-requisitos para participar?▾
Para cursos de nível inicial, não é necessário conhecimento prévio porque nós te guiamos desde os primeiros passos. Para outros, para que você tenha um bom aproveitamento é necessário conhecimento prévio em determinadas tecnologias. Consulte a página do treinamento para verificar o pré-requisito de cada um.
O treinamento dá direito ao exame?▾
Adquirindo o treinamento oficial, o participante ganha um voucher para realizar o exame. Quando apto na prova, conquista a credencial de certificação, seguindo os requisitos do fabricante.
Os treinamentos são online ou presenciais?▾
Prioritariamente online ao vivo. Para empresas, é possível agendar turmas fechadas exclusivas, online ou presenciais na sua região.
Qual a diferença entre os cursos?▾
A trilha de treinamento Zabbix é um programa desenvolvido para participantes com diferentes habilidades e necessidades. Porém, existem requisitos para alguns cursos — por exemplo, se você deseja receber o certificado de Profissional Certificado Zabbix, é obrigatório que você tenha concluído o curso de Especialista Certificado Zabbix. Você pode ver os pré-requisitos na descrição de cada curso.
